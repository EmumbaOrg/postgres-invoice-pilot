# 2.2 Clone the Repository

To get started, clone the [PostgreSQL Solution Accelerator: Invoice Pilot](http://aka.ms/pg-byoac-repo/) repository directly onto your local machine.

1. Open a terminal and navigate to the directory where you want to download the repo.
2. Run the following command to clone the repository:

    ```bash
    git clone https://github.com/Azure-Samples/postgres-invoice-pilot.git
    ```

3. Navigate into the cloned project directory:

    ```bash
    cd postgres-invoice-pilot
    ```

4. Open the project in Visual Studio Code:

    ```bash
    code .
    ```

!!! tip "Leave Visual Studio Code open as you will be using it throughout the remainder of the workshop."
